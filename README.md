# Evo Poker Hand Strength Evaluator

Algorithm for sorting poker hands according to their strength.

## Requirements
nodejs >= 14 < 15


## Installation
```
chmod +x prepare.sh
./prepare.sh
```
--or--

Use the package manager npm.

```bash
npm install --production=false
npm run build
```

## Usage

```
chmod +x run.sh
./run.sh < test-cases.txt > solutions.txt
```
